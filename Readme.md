### Installing ABI on Mac
```
go get -u github.com/ethereum/go-ethereum
cd $GOPATH/src/github.com/ethereum/go-ethereum/
go build ./cmd/abigen
```
ref: https://geth.ethereum.org/docs/tools/abigen
